Get started in seconds

Docker is the easiest path. Or download the desktop app.

🐳 Docker Recommended 

docker compose up --build

Open http://localhost:8002

🐍 Local Python

pip install -r backend/requirements.txt
python run.py

Requires Python 3.10+

Or skip Python entirely — download the desktop app for Mac or Windows.
